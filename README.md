# Sistema de reservas para restaurantes

## Descripción de la aplicación
La app móvil de reservas facilita encontrar restaurantes cercanos y reservar en tiempo real desde el teléfono, para uso cotidiano (salir a comer, planificar reuniones o encontrar disponibilidad).
